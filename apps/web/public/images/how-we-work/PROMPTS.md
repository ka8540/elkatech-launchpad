# How We Work Image Prompts

The current `.webp` files are project-local AI-style industrial placeholders with the same filenames the app renders. Replace them with generated final assets at the same paths when ready.

Shared direction for all five:

- Premium industrial technology
- Dark navy / graphite / steel palette
- Cyan-blue accent lighting
- Subtle grid or technical blueprint overlays
- Realistic industrial printing/signage machinery environment
- Cinematic but clean
- No cartoon style
- No random people smiling at camera
- No text inside the image
- No logos unless the ElkaTech mark is intentionally used
- No fake UI text
- No watermark
